### Install & Run
```bash

pnpm install
pnpm dev
```

### Alternative Port
```bash
pnpm dev -- -p 3001
```

### Clean Install
```bash
rm -rf node_modules .next pnpm-lock.yaml
pnpm install
pnpm dev
```

### Production Build
```bash
pnpm build
pnpm start
```
